一、快速开始

狼人杀发牌员是一款支持跨设备同步的狼人杀游戏辅助工具。法官创建房间后，玩家通过房间号加入，所有操作实时同步，让线下狼人杀游戏更加便捷。

1.1 法官创建房间

1 打开应用

访问网址：https://xjqfcmek5qaew.ok.kimi.link

2 点击"创建房间"

选择"我是法官"进入房间创建页面

3 配置游戏设置

设置玩家人数、胜负模式、是否开启警长

4 分享房间号

将6位数字房间号告诉所有玩家

1.2 玩家加入房间

1 打开应用

使用与法官相同的网址

2 点击"加入房间"

选择"我是玩家"

3 输入房间号和名字

输入法官提供的6位房间号，设置你的名字（建议格式：位置+姓名）

进入房间后仍可修改名字，点击名字旁边的编辑图标即可。

二、法官操作指南

2.1 创建房间设置

玩家人数

支持6-20人游戏，法官不计入玩家人数。根据人数会自动推荐合适的板子配置。

胜负模式

模式 狼人胜利条件 好人胜利条件
屠边 杀死所有神职或所有平民 投票投出所有狼人
屠城 杀死所有好人（神职+平民） 投票投出所有狼人

警长竞选

8人以上默认开启警长，但法官可以手动关闭。警长拥有1.5票投票权和归票权。

角色配置

系统会根据人数和胜负模式自动推荐板子，法官也可以选择"自定义"调整角色配置。

角色总数必须与玩家人数相等才能开始发牌。

2.2 管理玩家

添加玩家

在等待阶段，法官可以在玩家列表底部输入玩家名字并点击"+"按钮添加玩家。

修改玩家名字

点击玩家名字旁边的编辑图标（✏️），输入新名字后按回车或点击确认保存。

踢出玩家

点击玩家旁边的删除图标（❌），确认后即可将玩家踢出房间。被踢出的位置可以让新玩家补位。

踢人功能仅在等待阶段可用，游戏开始后无法踢人。

2.3 发牌与游戏控制

开始发牌

当所有玩家加入后，点击底部"开始发牌"按钮。系统会随机分配角色给所有玩家。

查看玩家角色

发牌后，法官可以在玩家列表中看到每个玩家的角色。玩家只能看到自己的角色。

标记玩家状态

游戏进行中，点击玩家旁边的骷髅图标可以标记玩家死亡/存活状态。

警长管理

在"玩家"标签页中，可以：

  • 点击皇冠图标颁发警徽
  • 点击撕毁图标撕毁警徽
  • 警长出局时会提示传递警徽

重新开始

游戏结束后，法官可以点击"重新开始"清空所有角色，回到等待阶段。

2.4 法官记录面板

在游戏进行中，法官可以切换到"记录"标签页，记录每晚的行动和白天的投票。

夜晚记录

点击"记录夜晚"按钮，依次记录：

  1. 守卫守护：选择守卫守护的目标（如有守卫）
  2. 狼人击杀：选择狼人击杀的目标
  3. 女巫操作：选择是否使用解药/毒药
  4. 预言家查验：选择查验目标和结果
女巫解药和毒药各只有一瓶，使用后会自动标记为已用完。

白天记录

点击"记录白天"按钮，记录：

  1. 特殊行动：白狼王自爆、骑士决斗、狼枪开枪等
  2. 被投出局：选择被投票出局的玩家
  3. 投票详情：记录每位玩家的投票对象

游戏结束判断

系统会根据胜负模式自动判断游戏是否结束，并提示获胜阵营。

三、玩家操作指南

3.1 加入房间

玩家加入房间的步骤：

  1. 打开应用网址
  2. 选择"我是玩家"
  3. 输入6位房间号
  4. 输入你的名字（建议格式：位置+姓名，如"1号张三"）
  5. 点击"加入房间"
如果房间已满，请联系法官踢出一个玩家后再加入。

3.2 查看身份

法官发牌后，玩家可以在"玩家"标签页看到自己的角色卡片：

  1. 点击"翻开身份"按钮查看角色
  2. 角色卡片显示：角色名称、阵营（好人/狼人）、技能说明
  3. 点击卡片可以盖回/翻开
  4. 切换到"提示"标签页查看角色游戏提示
角色身份仅自己可见，请勿将手机展示给其他玩家！

3.3 修改名字

进入房间后如果需要修改名字：

  1. 在玩家列表中找到自己的名字
  2. 点击名字旁边的编辑图标（✏️）
  3. 输入新名字
  4. 按回车或点击确认保存
建议名字格式：位置+姓名（如"1号张三"），方便法官辨认。

退出房间

点击左上角的"退出"按钮，确认后即可退出房间。退出后可以重新加入。

五、常见问题

Q1: 无法创建房间怎么办？

请检查网络连接，确保能正常访问互联网。如果问题持续，请刷新页面重试。

Q2: 玩家加入时提示"房间已满"？

法官可以踢出一个等待中的玩家，或者重新开始一局游戏。

Q3: 发牌后玩家看不到角色？

请玩家切换到"玩家"标签页，点击"翻开身份"按钮查看角色卡片。

Q4: 女巫解药用完了还能知道吗？

不能。女巫解药用完后，法官不会再告知当晚的死者。

Q5: 守卫和女巫同时保护同一人会怎样？

同守同救会导致该玩家死亡（奶穿）。

Q6: 游戏进行中可以添加新玩家吗？

不可以。只能在等待阶段添加玩家，游戏开始后无法添加。

Q7: 法官退出后房间会怎样？

法官退出后，房间会关闭，所有玩家都会被断开。建议法官在游戏结束前不要退出。

Q8: 如何修改已设置的板子？

在等待阶段，法官可以点击"角色配置"区域的"自定义"按钮调整角色配置。

如有其他问题，请联系法官或重新创建房间。
